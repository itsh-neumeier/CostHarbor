# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-03-07

### Added

- Initial MVP release
- Admin authentication with bcrypt password hashing
- Site, unit, and tenant management
- Recurring cost items with area-based allocation
- Water split rules (configurable, default 50/50)
- Home Assistant API integration with entity mapping
- Shelly Pro 3EM CSV import
- VRM IMAP email fetch with download link extraction
- Manual VRM CSV upload fallback
- aWATTar dynamic hourly price import
- Monthly billing calculation engine
- Billing preview in the web UI
- German-language PDF invoice generation (WeasyPrint)
- Audit logging for all administrative changes
- Versioned configuration and calculation runs
- Docker and docker-compose setup
- GitHub Actions CI/CD pipelines
- Dependabot security scanning
